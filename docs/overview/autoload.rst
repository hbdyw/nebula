********************
默认目录
********************

新安装程序都有 4 个目录：\ ``/admin``\ ,\ ``/content``\ ,\ ``/includes``\ ,\ ``/docs``\ ，这些目录每个都有非常具体的使用规范。

admin
====================

存放后台文件与一部分静态资源，如果你是新手，勿动，详情请见后续说明。

content
====================

内容文件夹，插件、主题、上传文件等存放于此。

includes
====================

系统文件，如果你是新手，勿动，详情请见后续说明。


docs
====================

程序文档，可直接打开里面 ``index.html`` 查看文档。
